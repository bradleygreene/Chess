## General Information
Chess is ran with:
- Python 2.7.17
- Pygame 1.9.6
### Controls
To move a piece, simply click on the piece you want to move and the 
location you want it to go.

To undo a move, press `Z`
## Setup
You can run the program by either :
1. Command line
   - `$ python ChessMain.py`
2. PyCharm
   - running ChessMain.py
